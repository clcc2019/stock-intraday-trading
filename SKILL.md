# 股票趋势分析与日内做T

## 投资哲学

### 两条公设
1. **价格围绕价值波动** — 均线是价值中枢，价格偏离后必然回归
2. **钟摆式过度波动** — 价格因贪婪恐惧过度摆动，偏离越远回归力越大

### 核心原则
- **内功为本招式为辅** — 基本面定方向，技术面把时机
- **顺大势逆小势** — 顺应大趋势，利用小回调进出
- **均线 = 价值中枢** — 均线方向=价值方向，价格与均线距离=钟摆位置

### 评分体系（100分）
- **基本面50分**：盈利(15)+成长(10)+健康(10)+估值(10)+资金(5)
- **技术面50分**：趋势(15)+钟摆(12.5)+强度(10)+量价(7.5)+指标(5)

### 均线层级与多级别钟摆
| MA250(年线) | MA120(半年) | MA60(季线) | MA20(月线/中枢) | MA10(短期情绪) | MA5(超短期) |

### 多级别钟摆模型
- **MA5偏离**：超短期情绪（>5%过热，<-3%超跌）
- **MA10偏离**：短期情绪（>4%过热，<-3%超跌）
- **MA20偏离**：中期中枢（>8%追高风险大，<-8%反弹动力强）
- **最佳买点**：MA5/MA10/MA20三者收敛（均线簇收敛）
- **高位风险**：MA5>5%且MA20>8%时避免追高

### 见顶/出货检测（趋势末端识别）
场景：MA20仍向上但短期价格连续下跌——需判断行情是否结束、主力是否出货

| 信号维度 | 权重 | 说明 |
|---------|------|------|
| MA5连续下行 | 20分 | 短线资金持续撤离，MA5下穿MA10=短线死叉 |
| 连续阴线 | 15分 | 卖压持续，连续3+根阴线=明显抛压 |
| 量价背离 | 20分 | 价格在高位(距高点5%内)但量能萎缩=拉高出货 |
| 放量滞涨/下跌 | 15分 | 放量但价不涨或下跌=主力对倒出货 |
| MACD顶背离 | 15分 | 价格新高但DIF走低=动量衰减 |
| 长上影线 | 10分 | 上方抛压重，近3日多根长上影=见顶K线 |
| 短长期分歧 | 5分 | MA5死叉但远离MA20=趋势末端分歧 |
| 高位缩量回落 | 15分 | 从近期高点回落>3%+量缩至<60%+连跌2日=资金撤退 |

- **>=70分**：危险（选股直接排除，分析降级为卖出）
- **>=50分**：警惕（选股强度-3，做T标记警告，分析降级为观望）
- **>=30分**：注意（选股强度-2，做T标记警告）
- **<30分**：安全

### 底部反弹检测（价值回归识别）
场景：基本面优秀的股票经历较大跌幅后出现技术面止跌企稳信号——"跌够了的好股票"

**三层筛选逻辑：**
1. **基本面过滤**：价值评分>=5/10，排除价值陷阱（低ROE+亏损）
2. **跌幅过滤**：价格<MA60 | MA20偏离<-3% | 60日跌幅>=15%
3. **底部信号**：技术面出现止跌/反转特征

| 信号维度 | 权重 | 说明 |
|---------|------|------|
| RSI超卖 | 15分 | RSI<30=超卖，<20=极度超卖，反弹动力强 |
| KDJ超卖金叉 | 15分 | K/D<20+J<0=超卖区，低位金叉=反转信号 |
| MACD底背离 | 20分 | 价格新低但DIF不新低=下跌动能衰竭（强信号） |
| 缩量企稳 | 15分 | 地量见地价：量能极度萎缩+价格止跌=抛压枯竭 |
| 布林带下轨 | 10分 | 价格触及/跌破下轨后反弹=超跌反弹 |
| 均线金叉 | 15分 | MA5低位金叉MA10、均线粘合=即将变盘 |
| 反转K线 | 10分 | 长下影线、十字星、早晨之星=底部形态 |

- **>=60分**：强底部信号（优先推荐）
- **>=40分**：中等底部信号（需关注确认）
- **>=25分**：弱底部信号（纳入候选）
- **<25分**：无信号（排除）

**价值评估（10分制）：** ROE质量(0-3) + PE低估(0-3) + 营收增长(0-2) + ROE+PE联合(0-2)
**排除价值陷阱：** ROE<5% | 营收下滑>10% | PE为负（亏损）

## AI执行指令

### 1. 意图识别
- "选股/筛选/推荐" → 趋势选股脚本（默认 `--strategy trend`）
- "底部/反弹/超跌/低估/抄底/被低估/错杀" → 底部反弹选股（`--strategy bottom`）
- "做T/日内/分时" → T+0脚本
- "回测/历史收益" → 回测脚本
- "图表/K线图" → 图表生成脚本
- 其他 → 日线分析脚本

### 2. 代码识别
常见：贵州茅台(600519)、比亚迪(002594)、恒瑞医药(600276)、宁德时代(300750)

### 3. 执行脚本

**趋势选股**（追涨不追高，默认策略）：
```bash
cd .cursor/skills/stock-intraday-trading && python3 scripts/select_stocks.py
# 默认使用核心指数(沪深300+上证50,约320只)，首次约60s，缓存后约10s
# --index 选项: core(默认), hs300, zz500, sz50(最快), wide(沪深300+中证500), all(全A股)
# --no-fundamental   跳过基本面（纯技术面筛选更快）
# --top 30           显示前N只（默认30）
```

**底部反弹选股**（基本面好+跌够了+底部信号）：
```bash
cd .cursor/skills/stock-intraday-trading && python3 scripts/select_stocks.py --strategy bottom
# 寻找基本面优秀但被低估、即将触底反弹的股票
# --strategy bottom   启用底部反弹策略
# --index wide        建议用宽基指数，底部股票在中小盘更多
# --no-fundamental    跳过基本面（纯技术面底部信号）
```

**做T分析**：
```bash
cd .cursor/skills/stock-intraday-trading && python3 scripts/analyze_intraday_t0.py 股票代码
```

**日线分析**：
```bash
cd .cursor/skills/stock-intraday-trading && python3 scripts/analyze_stock_simple.py 股票代码
```

**回测**：
```bash
cd .cursor/skills/stock-intraday-trading && python3 scripts/backtest_strategy.py 股票代码
```

**图表**：
```bash
cd .cursor/skills/stock-intraday-trading && python3 scripts/generate_chart_data.py 股票代码 --open
```
