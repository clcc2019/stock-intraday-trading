# 股票趋势分析与日内做T

## 投资哲学（核心理念）

本工具基于以下投资哲学构建，所有分析策略均由此推导：

### 两条公设

1. **价格围绕价值波动**（人牵狗）— 均线是价值中枢在图表上的投影，价格偏离均线后必然回归
2. **钟摆式过度波动** — 价格因贪婪和恐惧过度摆动，偏离越远，回归力越大

### 核心原则

- **内功为本，招式为辅** — 基本面（内功）决定价值方向，技术面（招式）把握时机
- **顺大势，逆小势** — 顺应大级别趋势方向，利用小级别回调/反弹进出
- **趋势 = 方向 + 强度** — 均线多头排列=方向向上，相对强度/持续性=势的大小
- **均线 = 价值中枢** — 均线方向=价值方向，价格与均线距离=情绪偏离度（钟摆位置）

### 评分体系（100分制）

- **基本面（内功）50分**：盈利能力(15) + 成长能力(10) + 财务健康(10) + 估值水平(10) + 资金面(5)
- **技术面（招式）50分**：趋势方向(15) + 钟摆位置(12.5) + 趋势强度(10) + 量价关系(7.5) + 传统指标(5)

### 分析层级

| 层级 | 对应均线 | 含义 |
|------|---------|------|
| 超长期（年线） | MA250 | 长期价值方向 |
| 长期（半年线） | MA120 | 中长期趋势 |
| 中期（季线） | MA60 | 中期趋势 |
| 短期（月线） | MA20 | 短期趋势/价值中枢 |
| 超短期 | MA5/MA10 | 短期情绪波动 |

## ⚠️ AI助手执行指令

当用户询问股票分析或买卖建议时：

### 必须执行

1. **识别意图**
   - 如用户提到"选股"、"筛选"、"推荐股票"、"趋势向上" → 使用选股脚本
   - 如用户提到"做T"、"日内"、"分时"、"T+0" → 使用T+0脚本
   - 如用户提到"回测"、"历史收益"、"策略验证" → 使用回测脚本
   - 如用户提到"图表"、"可视化"、"K线图"、"看图" → 使用图表数据生成脚本
   - 否则使用日线分析脚本

2. **识别股票代码**
   - 常见代码：贵州茅台(600519)、比亚迪(002594)、恒瑞医药(600276)、宁德时代(300750)
   - 如用户只提供名称，先转换为6位代码

3. **执行对应脚本**
   
   **趋势选股**（筛选趋势向上+基本面好的股票）：
   ```bash
   # 从沪深300中选股（推荐，含基本面评分）
   cd .cursor/skills/stock-intraday-trading && python3 scripts/select_stocks.py --index hs300
   
   # 从中证500中选股
   cd .cursor/skills/stock-intraday-trading && python3 scripts/select_stocks.py --index zz500
   
   # 指定板块选股
   cd .cursor/skills/stock-intraday-trading && python3 scripts/select_stocks.py --sector 白酒
   
   # 跳过基本面分析（加速选股，仅看技术面）
   cd .cursor/skills/stock-intraday-trading && python3 scripts/select_stocks.py --index hs300 --no-fundamental
   
   # 全A股选股（较慢，需要较长时间）
   cd .cursor/skills/stock-intraday-trading && python3 scripts/select_stocks.py
   ```
   
   **T+0日内做T分析**：
   ```bash
   cd .cursor/skills/stock-intraday-trading && python3 scripts/analyze_intraday_t0.py [股票代码]
   ```
   
   **日线综合分析**：
   ```bash
   cd .cursor/skills/stock-intraday-trading && python3 scripts/analyze_stock_simple.py [股票代码]
   ```
   
   **策略历史回测**：
   ```bash
   cd .cursor/skills/stock-intraday-trading && python3 scripts/backtest_strategy.py [股票代码]
   cd .cursor/skills/stock-intraday-trading && python3 scripts/backtest_strategy.py --multi
   ```

   **可视化图表分析**（生成数据 → 打开交互式K线图）：
   ```bash
   # 生成图表数据并自动打开浏览器
   cd .cursor/skills/stock-intraday-trading && python3 scripts/generate_chart_data.py [股票代码] --open

   # 仅生成数据（不打开浏览器）
   cd .cursor/skills/stock-intraday-trading && python3 scripts/generate_chart_data.py [股票代码]

   # 指定显示天数
   cd .cursor/skills/stock-intraday-trading && python3 scripts/generate_chart_data.py [股票代码] --days 200 --open
   ```
   > 图表页面固定存放在 `chart/stock-chart.html`，每次仅更新数据文件 `chart/stock_data.json`

4. **展示结果**
   - 直接展示脚本完整输出（已包含基本面+技术面融合分析）
   - 可选：用1-2句话总结核心建议
   - 提醒用户关注定性因素（管理层、公司文化、行业竞争格局）
