# 量价关系实战指南

## 核心理念

量是因，价是果。成交量变化先于价格变化，是资金流向的直接体现。

---

## 经典量价模式

### 1. 放量上涨 📈💰

**特征**:
- 价格上涨 + 成交量显著放大（量比 > 1.5）
- 买盘力量强劲

**含义**:
- **初期放量上涨**: 启动信号，积极买入
- **持续放量上涨**: 趋势延续，持有
- **高位放量上涨**: 可能是最后的狂欢，注意风险

**操作建议**:
```
✅ 低位放量突破 → 买入
✅ 中期放量上涨 → 持有
⚠️ 高位巨量上涨 → 减仓（可能是顶部）
```

---

### 2. 缩量上涨 📈📉

**特征**:
- 价格上涨 + 成交量萎缩（量比 < 0.8）
- 上涨动能不足

**含义**:
- **缩量突破**: 真突破还是假突破？需验证
- **缩量新高**: 空方力量弱，但多方也不积极
- **可持续性**: 较弱，容易回调

**操作建议**:
```
⚠️ 观望为主，等待放量确认
✅ 如已持仓，逢高减仓
❌ 不建议追高
```

---

### 3. 放量下跌 📉💰

**特征**:
- 价格下跌 + 成交量放大
- 恐慌性抛售

**含义**:
- **高位放量下跌**: 主力出货，危险信号
- **低位放量下跌**: 可能是最后的杀跌，关注止跌信号
- **跌停板放量**: 极度恐慌，但可能是底部

**操作建议**:
```
❌ 高位放量下跌 → 止损离场
⚠️ 中位放量下跌 → 减仓观望
✅ 低位放量下跌 → 等待企稳信号（如出现缩量或反弹）
```

---

### 4. 缩量下跌 📉📉

**特征**:
- 价格下跌 + 成交量萎缩
- 抛压减轻

**含义**:
- 下跌动能衰竭
- 可能止跌企稳
- 空方力量减弱

**操作建议**:
```
✅ 关注止跌信号
⏳ 等待放量反弹确认
✅ 可尝试小仓位试探
```

---

### 5. 缩量横盘 ➡️📉

**特征**:
- 价格窄幅波动 + 成交量持续萎缩
- 多空双方力量平衡

**含义**:
- **低位缩量盘整**: 主力吸筹，耐心等待
- **高位缩量盘整**: 出货后期，观望
- **中位缩量盘整**: 变盘在即

**操作建议**:
```
⏳ 观望等待
✅ 低位关注放量突破向上
⚠️ 高位关注放量突破向下
```

---

### 6. 放量横盘 ➡️💰

**特征**:
- 价格横盘 + 成交量放大
- 换手充分

**含义**:
- **低位放量横盘**: 主力建仓，看涨
- **高位放量横盘**: 出货，看跌
- **变盘在即**: 即将选择方向

**操作建议**:
```
✅ 低位放量盘整 → 积极关注，突破买入
⚠️ 高位放量盘整 → 警惕出货，减仓
```

## 量价背离

### 顶背离（看跌）

**特征**:
- 价格创新高
- 成交量不创新高（甚至萎缩）

**含义**: 上涨乏力，顶部信号

**操作**:
```
⚠️ 减仓
❌ 不追高
✅ 设置止损
```

### 底背离（看涨）

**特征**:
- 价格创新低
- 成交量不创新低（或明显萎缩）

**含义**: 下跌动能衰竭，底部信号

**操作**:
```
✅ 关注企稳信号
✅ 分批建仓
⏳ 等待放量反弹确认
```

## 实战案例

### 案例 1: 地量地价（强买入）

```
情景:
- 股价经过长期下跌，跌幅 > 50%
- 成交量极度萎缩至历史低位（量比 < 0.3）
- 出现止跌企稳迹象

判断:
- 抛压释放完毕
- 筹码锁定良好
- 底部信号明确

操作:
✅ 分批建仓
✅ 等待放量突破确认
```

### 案例 2: 天量天价（强卖出）

```
情景:
- 股价已大幅上涨（涨幅 > 100%）
- 突然出现历史天量（量比 > 5）
- 当日冲高回落或第二天低开

判断:
- 主力借放量出货
- 短期顶部形成
- 获利盘集中涌出

操作:
❌ 立即止盈
⚠️ 不参与反弹
```

### 案例 3: 突破放量（趋势启动）

```
情景:
- 股价长期横盘整理（3个月+）
- 突然放量突破关键压力位（量比 > 2）
- 收盘站稳突破位上方

判断:
- 主力开始拉升
- 趋势启动信号
- 上涨空间打开

操作:
✅ 突破买入
✅ 持股待涨
✅ 设置移动止损
```

### 案例 4: 缩量回调（健康调整）

```
情景:
- 股价上涨途中出现回调
- 回调时成交量明显萎缩（量比 < 0.5）
- 回调至关键支撑位（如 MA10）企稳

判断:
- 主力惜售，筹码稳定
- 健康洗盘
- 上涨趋势未改

操作:
✅ 支撑位加仓
✅ 持股不动
```

## 实战技巧

### 1. 量比监控

实时监控量比变化：

```python
def monitor_volume_ratio(stock_code):
    """实时监控量比"""
    data = get_stock_data(stock_code)
    volume_ratio = data['volume'] / data['avg_volume_20']
    
    if volume_ratio > 2:
        print("🔥 成交量异常放大，关注资金动向")
    elif volume_ratio < 0.5:
        print("📉 成交量极度萎缩，关注止跌信号")
    else:
        print("✅ 成交量正常")
```

### 2. 量能堆积

连续多日成交量递增：
- 上涨 + 量能堆积 = 强势上涨
- 下跌 + 量能堆积 = 恐慌杀跌

### 3. 均量线

类似均价线，观察成交量趋势：
- 5日均量线上穿 10日均量线 = 量能放大
- 5日均量线下穿 10日均量线 = 量能萎缩

## 注意事项

1. **量价配合才有效**: 单看价格或单看成交量都不够
2. **结合趋势判断**: 同样的量价关系在不同趋势中意义不同
3. **关注换手率**: 高换手 + 放量 = 强烈的多空分歧
4. **警惕对倒**: 主力可能通过对倒制造虚假成交量
5. **大盘影响**: 个股量价需结合大盘整体环境
