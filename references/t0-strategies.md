# 日内T+0做T策略详解

## 什么是T+0做T？

**T+0做T** = 利用已有底仓，在同一交易日内**低买高卖**或**高卖低买**，实现日内套利，降低持仓成本。

---

## 投资哲学框架

所有做T策略都基于以下哲学：

### 两条公设

1. **价格围绕价值波动** → 日内价格围绕VWAP（日内价值中枢）波动
2. **钟摆式过度波动** → 日内涨跌会过头，偏离VWAP越远，回归力越大

### 做T的嵌套钟摆模型

| 级别 | 对应 | 做T含义 |
|------|------|---------|
| 大势 | 周线/日线MA方向 | 决定做T方向（多/空/双向） |
| 小势 | 日内分时波动 | 利用日内波动进出 |
| 价值中枢 | VWAP + 日线均线 | 做T的核心参考线 |
| 钟摆位置 | 价格偏离VWAP/MA的程度 | 做T的买卖依据 |

### 核心原则：顺大势，逆小势

- **顺大势** = 只在日线/周线趋势向上的股票做T买入
- **逆小势** = 利用日内回调（小钟摆回摆）买入，利用日内冲高（小钟摆偏离）卖出

---

## 策略总纲：顺大势逆小势

### 做T方向决策

| 大势（日线/周线） | 做T方向 | 操作要点 |
|-------------------|---------|---------|
| 趋势向上（强度≥6） | 偏多做T | 日内回踩VWAP买入为主，冲高卖出 |
| 趋势向上（强度4-5） | 谨慎做T | 轻仓操作，严格止损 |
| 震荡 | 双向做T | 高卖低买，区间操作 |
| 趋势向下 | ⚠️不建议 | 趋势向下做T买入风险极高 |

### 做T的前提条件

1. ✅ 了解该股票趋势向上的基本面原因（内功）
2. ✅ 日线/周线均线多头排列
3. ✅ 趋势强度评分≥4/10
4. ✅ 日内波动幅度≥2%（有做T空间）
5. ❌ 大盘弱势（上证跌幅>1.5%）时停止做T

---

## 核心策略1: 趋势+回踩VWAP做T ⭐⭐⭐⭐⭐

**最核心的做T策略，直接基于两条公设推导**

### 原理

- VWAP（日内成交均价，黄线）= 日内价值中枢
- 价格回踩VWAP = 小钟摆回摆到静止点 = 买入时机
- 价格偏离VWAP = 小钟摆偏离 = 卖出时机

### 买入条件

1. 大势看多（日线均线多头排列，趋势强度≥6）
2. 日内价格回踩至VWAP下方（偏离-0.3%以上）
3. 回踩过程缩量（恐慌情绪释放）

### 卖出条件

1. 日内价格回升至VWAP上方（偏离+0.5%以上）
2. 或达到目标收益（+1~2%）

### 实战案例

```
某股日线均线多头排列，趋势强度8/10，开盘¥50.00，VWAP ¥50.20

10:00 - 价格回踩至¥49.85（VWAP下方-0.7%）→ 买入
11:00 - 价格回升至¥50.50（VWAP上方+0.6%）→ 卖出

收益: (50.50 - 49.85) / 49.85 = +1.3%
```

## 核心策略2: 均线偏离度均值回归 ⭐⭐⭐⭐⭐

### 原理

- 公设一：价格偏离均线后必然回归
- 公设二：偏离越远，回归力越大
- 当日线价格大幅偏离MA20/MA60时，均值回归概率极高

### 做T应用

**偏离MA20过大时（>+8%）**：
- 日内冲高 → 做T卖出（绳子太紧，回归压力大）
- 等价格向MA20方向回调后再买入

**回踩MA20附近时（-3%~+3%）**：
- 日内回踩 → 做T买入（接近价值中枢，钟摆回摆）
- 这是「顺大势逆小势」的理想买点

### 偏离度与操作策略

| 偏离MA20 | 钟摆位置 | 做T策略 |
|---------|---------|---------|
| > +10% | 极度偏高 | 逢高必卖，不做买入 |
| +5% ~ +10% | 偏高 | 以卖出为主 |
| -3% ~ +5% | 中枢附近 | 正常做T，双向操作 |
| -5% ~ -3% | 偏低 | 以买入为主（需大势看多） |
| < -5% | 偏低 | 积极买入（需大势看多 + 基本面确认） |

## 核心策略3: 趋势强度过滤（风控）⭐⭐⭐⭐

### 原理

势大力沉 — 趋势越强，做T的安全边际越高。

### 过滤规则

| 趋势强度 | 做T建议 | 仓位 |
|---------|---------|------|
| 8-10/10 | 积极做T | 1/3仓位 |
| 6-7/10 | 正常做T | 1/4仓位 |
| 4-5/10 | 谨慎做T | 1/5仓位 |
| 0-3/10 | 不做T | 0 |

---

## 辅助策略4: 分时均价线策略 ⭐⭐⭐⭐

分时均价线（VWAP，黄线）= 日内价值中枢

| 价格位置 | 操作 | 理由 |
|---------|------|------|
| 跌破均价线 -0.5% | 🟢 买入做T | 价格低于日内价值中枢，超跌 |
| 回归均价线附近 | 🟡 观望 | 价格回到正常水平 |
| 突破均价线 +0.5% | 🔴 卖出做T | 价格高于日内价值中枢，超买 |

---

## 辅助策略5: 支撑压力位策略 ⭐⭐⭐⭐

### 关键价位（以均线为核心）

**支撑位**（买入参考）：
- MA20（日线月均线）— 最重要的支撑
- MA60（日线季均线）— 中期支撑
- 昨日最低价
- 整数关口

**压力位**（卖出参考）：
- 昨日最高价
- 前期高点
- 均线上方密集区

### 操作方法

1. 开盘确认支撑/压力位
2. 价格触及支撑位（±0.5%）→ 买入
3. 价格触及压力位（±0.5%）→ 卖出
4. 设置止损（支撑位下破-1%离场）

---

## 辅助策略6: 时间窗口策略 ⭐⭐⭐

| 时间段 | 特征 | 策略 |
|--------|------|------|
| 9:30-10:00 | 开盘波动大 | 观望为主 |
| 10:00-10:30 | 方向确立 | 顺势做T |
| 10:30-11:30 | 午前高点/低点 | 反向做T |
| 13:00-14:00 | 午后开盘 | 观察走势 |
| 14:00-14:30 | 日内高点易出现 | 准备卖出 |
| 14:30-15:00 | 尾盘 | 锁定利润 |

### 经典操作

**早盘急跌抄底**（需大势看多）：
- 条件：9:30-10:30，跌幅>-2%，大盘正常，日线趋势向上
- 操作：分批买入，目标回补至VWAP
- 原理：大势向上+小钟摆过度向下=均值回归机会

**午后冲高获利**：
- 条件：13:00-14:30，涨幅>+2%，价格位置>70%
- 操作：卖出获利
- 原理：小钟摆偏高，锁定利润

## T+0操作纪律

### 仓位管理

| 波动幅度 | 做T仓位 | 交易频率 | 单次目标 |
|---------|---------|---------|---------|
| 日内振幅 > 4% | 1/3仓位 | 2-3次 | +1.5-2% |
| 日内振幅 2-4% | 1/4仓位 | 1-2次 | +1-1.5% |
| 日内振幅 < 2% | 观望 | 0次 | - |

### 止损止盈

- ✅ **严格止损**: 单次亏损 -1% 立即止损
- ✅ **及时止盈**: 达到目标收益即止盈
- ✅ **快进快出**: 日内交易，不过夜

### 大盘环境

| 大盘涨跌 | T+0策略 |
|---------|---------|
| > +1.5% | 积极做多T |
| +0.5% ~ +1.5% | 正常做T |
| -0.5% ~ +0.5% | 谨慎做T |
| < -1.5% | 停止做T |

## 内功与招式

### 招式（技术面 — 做T操作）

做T的技术要点：
1. 趋势方向 → 决定做T方向
2. 均线偏离度 → 决定买卖时机
3. VWAP → 日内价值中枢
4. 量能变化 → 确认信号

### 内功（基本面 — 选股基础）

做T的前提是选对股票：
- 你必须了解这只股票趋势向上的原因
- 业绩增长？行业景气？政策利好？
- 是真实的价值提升，还是资金炒作？
- 有没有潜在风险？

**记住**：技术分析只能看到「狗」（价格），看不到「人」（价值）。只有理解了基本面，你才能在做T时有信心持有底仓、敢于逢低买入。

## 附录：MACD/KDJ辅助参考（可选）

> 以下指标本质上是均线偏离度的衍生计算，均线分析已涵盖其核心信息。
> 仅作为额外参考，不作为核心决策依据。

### MACD(8,17,9)

- 金叉 = 短期均线加速上穿长期均线（相当于偏离度从负转正）
- 死叉 = 短期均线加速下穿长期均线（相当于偏离度从正转负）
- 红绿柱 = 短期与长期均线的差值变化速度

### KDJ(6,3,3)

- J > 80 = 价格在近期区间高位（相当于偏离均线过高）
- J < 20 = 价格在近期区间低位（相当于偏离均线过低）
- 金叉/死叉 = 价格在区间内的位置变化方向

## 风险提示

### ❌ 常见错误

1. **频繁交易**：一天10+次，手续费吃掉利润
2. **逆势做T买入**：大势向下还做多
3. **重仓做T**：一次性用全部仓位
4. **不设止损**：亏损扩大到-5%+
5. **不了解基本面**：只看图表，不知道为什么涨

### ✅ 成功要点

1. **顺大势**：只在趋势向上的股票做T买入
2. **逆小势**：利用日内回调/冲高做T
3. **严格止损**：单次-1%止损
4. **控制仓位**：1/4-1/3仓位
5. **了解基本面**：知道趋势向上的原因
