# 投资经验原文整理


## 第一章：传统技术分析的真相 — 巫术的破灭

### 残酷的真相

绝大多数人痴迷的传统技术分析 — K线形态（早晨之星、乌云盖顶）、技术指标（MACD、KDJ）、玄学理论（波浪理论、江恩理论、缠论）— 在今天这个时代，**通通都是垃圾**。

这并非危言耸听。要理解这一点，必须从历史讲起。

### 技术分析的历史起源

所有技术分析 — 无论是东方的日本蜡烛图（酒田战法），还是西方的道氏理论 — 都诞生于**19世纪末20世纪初**。

那是一个没有电脑、没有互联网、甚至没有计算器的时代。如果你是100年前的交易员，你唯一拥有的实时、公平的数据，就是**价格**，最多再加上粗略的成交量。

在那种信息荒漠里，查尔斯·道和日本米商做了什么？他们做了当时看来天才一般的事情 — **从价格中寻找模式**：
- 道氏理论说「市场会消化一切」「平均指数必须相互验证」
- 日本米商说「三只乌鸦是下跌信号」

**技术分析的本质 = 量化投资的史前形态**
- 它和今天最顶尖的量化基金试图做的事情一模一样：在历史数据中寻找可重复出现的、能带来盈利的统计学模式
- 头肩顶 = 人脑识别的看跌模式
- 金叉 = 人脑定义的看涨模式
- 100年前，这是科学，是天才

### 计算机时代的两次演进

**第一阶段：计算机辅助（七八十年代）— 量化2.0**
- 用电脑代替人脑计算 → MACD、KDJ、RSI的由来
- 把「看形态」这种模糊归纳，变成「算指标」这种精确计算
- 这是今天99%散户所能接触到的技术分析的顶峰

**第二阶段：算法的降维打击（90年代至今）— 真·量化投资**
- 华尔街的顶尖基金（文艺复兴、AQR、D.E.Shaw）用超级计算机、人工智能
- 在微秒级别上，把你知道的以及你根本不知道的**几百万种模式**，扔到几十年数据里回测
- 不仅限于价格数据 — 新闻、财报、甚至天气都能加进模型

### 扎心的问题

假设MACD金叉这个模式真的能稳定赚钱，你觉得是你先发现，还是超级计算机先发现？

答案当然是超级计算机。如果这个模式真的有效，计算机早在20年前就发现了它，然后用高频交易算法疯狂套利，直到这个模式的超额收益**完全消失**为止。

**结论**：
- 你知道的那些K线形态和技术指标 = 量化投资1.0的产物
- 它们之所以还在教科书上流传、"大师"还在教你 → 唯一原因是**它们已经失效了**
- 真正有效的模式 = 量化基金的最高机密，你永远不可能知道
- 你在用算盘对抗超级计算机
- 你花10年时间精通的，是上个世纪的巫术

## 第二章：技术分析的地基 — 两条公设

巫术死了，但真理永存。那些花哨招式失效了，但技术分析的地基还在。这个地基，是超级计算机也无法套利掉的。

### 公设一：价格围绕价值波动（人牵狗）

本杰明·格雷厄姆（巴菲特的老师）的著名比喻：

- **价值** = 一个在路上行走的人（主人）
- **价格** = 他牵着的一条狗
- **价值规律** = 脖子上的绳子

狗（价格）会忽前忽后跑来跑去：
- 时而兴奋地冲到主人前面很远 → **泡沫**
- 时而被路边的东西吸引，远远落后 → **低估**

但狗永远不可能脱离主人太远。无论它跑得多偏，它最终都必须回到主人身边，并跟随主人（价值）的方向前进。

**三个残酷的真相**：
1. **价值才是核心** — 投资中唯一重要的事情是判断「人」（价值）要往哪里走
2. **狗只是表象** — 价格K线只是狗的轨迹
3. **绳子必然存在** — 价格可以短期偏离价值，但不会永远偏离，最终会回归

**在图表上的体现**：
- 长周期均线（如年线250日均线）= 那个牵狗的人在图表上的样子
- 它代表市场在过去一年里对这家公司价值的平均共识
- 当价格大幅偏离年线时 → 绳子绷紧 → 必然产生强大的力量把价格拉回 → **均值回归**

### 公设二：价格的钟摆式过度摆动

如果价格老老实实跟着价值走，我们直接研究基本面不就行了？关键在于：**价格不会老老实实跟着价值走**。

钟摆模型：
- **价值** = 钟摆的静止点
- **价格** = 钟摆本身
- **市场情绪** = 驱动钟摆的外力

钟摆永远不会温柔地停在静止点：
- 涨的时候，总会**涨过头**（摆到最右）→ 贪婪，追涨
- 跌的时候，也总会**跌过头**（摆到最左）→ 恐惧，踩踏

为什么？因为驱动价格短期波动的不是理性的价值，而是非理性的人性 — 贪婪、恐慌、信息不对称、羊群效应。

**物理定律**：钟摆摆得越远（价格偏离价值越远），回归的力量（绳子的拉力）就越大。

### 两条公设的结合

- 公设一（人牵狗）→ 价格最终会回归价值（回归均线）
- 公设二（钟摆）→ 价格在回归过程中会因为情绪而过度波动

**技术分析的正确定位**：
- ❌ 不是预测未来 — 任何试图预测短期走势的行为都是巫术，都会被量化算法降维打击
- ✅ 是追踪和衡量 — 它只是一个仪表盘

技术分析只能帮你回答三个问题：
1. **判断位置（偏离程度）**：狗（价格）现在离人（价值）有多远？
2. **判断方向（趋势）**：钟摆现在朝哪个方向摆？
3. **判断情绪（极端位置）**：钟摆是否摆到了极端？超买还是超卖？

## 第三章：玄铁剑法 — 趋势 + 均线

基于两条公设，我们真正需要的武器简单到不敢相信：**趋势 + 均线**。

这套剑法是量化算法无法打败你的。为什么？因为它不追求短期预测，它追求的是长期仓位和风险管理，而且它需要内功的配合。

### 一、趋势

趋 = 方向，势 = 强度。

#### 方向（趋）

最朴素、最可操作的定义（放弃所有复杂的波浪和通道）：
- **上升趋势** = 上涨创新高，调整不创新低
- **下降趋势** = 反弹不创新高，下跌创新低
- **震荡** = 反复穿越均线，既不创新高也不创新低（钟摆能量耗尽，在价值中枢附近小幅摆动）

翻译为投资语言：
- 上升趋势 = 价值本身在提升，人在上山，均线在上移
- 下降趋势 = 价值本身在下降，人在下山，均线在下移
- 震荡 = 价值稳定，鉴于钟摆能量耗尽

**原则**：作为价值投资者，只选择上升趋势的股票。因为清晰的上升趋势同时告诉你基本面大概率在改善。

#### 强度（势）

顺**势**而为，不是顺**趋**而为 — 你需要顺应的不仅仅是方向，还有一个**足够强大的浪潮**。

如何定义势够大：
- **相对强度**：大部分股票下跌时，这只股票不跌甚至逆势上涨 → 势大
- **持续时间**：两三年在涨 vs 十几年一直在涨 → 后者势大
- **市值大小**：大盘股上涨需要更多资金推动，更难操纵 → 势更大
- **板块联动**：个股上涨带动板块行情，甚至产业链扩散 → 势更大

**势大力沉**：选择更大的强度参与方向交易 = 更可靠，失败概率更小。

### 二、均线 — 玄铁重剑

移动平均线看起来又钝又慢，但它蕴含着当前价值的全部信息。

**均线的本质** = 一定周期内的平均价格 = 市场对价值的平均共识 = **价值中枢**（那个牵狗的人）

#### 功能一：方向 = 价值中枢的移动方向

- 均线向上 → 人在往上走，市场对价值的共识在抬升，基本面在改善
- 均线向下 → 人在往下走，市场共识在下降，基本面在恶化
- 均线走平 → 人在休息，价值稳定

#### 功能二：距离 = 情绪的偏离程度

- 价格远高于均线 → 狗跑到人前面很远 → 过度乐观，涨过头，绳子绷紧，有回调压力
- 价格远低于均线 → 狗落在人后面很远 → 过度悲观，跌过头，绳子绷紧，有反弹动力

**你根本不需要MACD和KDJ**：
- MACD的红绿柱是什么？不就是价格与均线距离的另一种画法吗？
- KDJ的超买超卖是什么？不就是价格偏离均线程度的另一种计算吗？
- 那些指标只是把简单直观的道理包装得面目全非

### 三、趋势的级别 — 钟摆的嵌套结构

为什么看起来这么简单的道理，在真实市场上却复杂无比？因为**趋势是有级别的**，**钟摆是嵌套的**。

| 级别 | 对应 | 钟摆大小 |
|------|------|---------|
| 年线（如5年均线）| 超大钟摆的支点 | 长期价值 |
| 月线（如5月均线）| 大钟摆 | 中期趋势 |
| 周线（如5周均线）| 小钟摆 | 短期趋势 |
| 日线（如5日均线）| 微型钟摆 | 超短期情绪波动 |

所有这些层层嵌套的钟摆的复合运动，最终表现为市场走势。

**核心规律**：小钟摆的摆动，最终会服从大钟摆的方向。

就像人在火车上 — 无论你是往前跑还是往后跑，时间拉长一点，几乎对位置改变不起作用。把你带到另一个城市的关键因素，还是火车。

**翻译为投资语言**：短期趋势服从长期趋势。日线趋势可能是下跌的（小钟摆在回摆），但如果月线年线趋势是上升的（大钟摆向上），那么日线下跌大概率只是技术性回调，而不是趋势反转。

### 四、玄铁剑法的唯一心法 — 顺大势，逆小势

- **大势** = 长周期趋势（月线、年线）
- **小势** = 短周期趋势（日线、周线）
- **顺大势** = 顺应大钟摆方向，只买月线/年线昂首向上的股票（确保你买的是人在往上走的好公司）
- **逆小势** = 利用小钟摆的回调（不在最贪婪时追高，而在短期恐慌回调到大钟摆支点附近时买入）

**买点解读**：你是在大钟摆向上的惯性中，去接小钟摆向下的底部。

为什么这是建仓良机：
- 长期价值在上升（大钟摆向上）
- 短期价格在回调（小钟摆回摆）
- 价格向价值中枢靠拢
- 根据公设一，回调是暂时的
- 根据大钟摆的惯性，回调结束后反弹会更快

**陷阱**：不能在日线刚开始跌的时候就买入。因为公设二告诉我们 — 既然要跌，它就会跌过头，会击穿价值中枢（月线），跌到所有人都恐慌绝望时才会反弹。所以要**等**，等小钟摆真正摆到底部、开始回摆时再买入。而判断底部，光靠技术分析是不可能完成的 — 必须配合基本面分析。

## 第四章：致命局限 — 你只看得到狗

### 技术分析看不到价值

两条公设能推导出：
- 价格偏离均线的程度（狗离人多远）
- 价格正在往哪个方向运动（钟摆在哪个阶段）

但它们**无法告诉你**：
- 人（价值）为什么要往上走
- 这个人会不会突然掉头或猝死（价值突变）

**举例**：一只股票月线向上、日线回调 — 完全符合「顺大势逆小势」的买点。技术分析告诉你买。但技术分析无法告诉你：
- 它为什么涨？是业绩真的爆发，还是庄家炒作虚假概念？
- 上涨能持续吗？支撑上涨的因素还在吗？
- 会不会明天爆出黑天鹅（财务造假、CEO被抓）→ 价值当场死亡 → 均线急转直下？

### 唯一正道：内功为本，招式为辅

用80%的时间修炼内功（基本面研究）：
- 商业模式是什么？
- 护城河在哪里？
- 管理层是诚实还是在撒谎？
- 财务数据是真实的还是伪造的？
- 行业景气度如何？

这是一条慢路、难路、反人性的路，但这是唯一通向**把时间当朋友**的路。

## 第五章：完整的投资流程（四步法）

### 第一步：技术筛选 — 观察趋势（望远镜）

用技术分析快速筛选标的：从几千只股票中，筛选出那些**大趋势向上**（月线向上）且**小钟摆回摆到支点附近**（日线回调接近月线）的股票。

技术分析的作用 = **望远镜**，帮你快速缩小范围。

### 第二步：基本面研究 — 理解支点为什么移动（显微镜）

对筛选出的股票做深入的内功研究：
- 为什么均线在上移？是业绩增长、行业景气，还是政策利好？
- 这是真实的价值提升，还是虚假繁荣？
- 有没有潜在的风险因素，会导致「人」突然猝死？

基本面分析的作用 = **显微镜**，帮你验证技术信号，确保你买的不是空架子。

### 第三步：前瞻判断 — 预测钟摆能摆多远

在确认基本面扎实的前提下，做前瞻性判断：
- 价值提升的因素能持续多久？未来一年，还是一次性利好？
- 大钟摆向上的惯性还有多强？
- 行业是初期爆发还是末期狂欢？
- 这轮向上摆动的目标位大概在哪里？估值上限在哪里？

这一步 = 内功和招式的结合，形成完整的投资逻辑。

### 第四步：交易决策 — 在钟摆回摆时出手

在日线回调到月线附近，也就是小钟摆回摆出现企稳信号时买入。

- 基本面分析（内功）告诉你**买什么**
- 技术分析（招式）告诉你**何时买**
- 两者的关系不是对立，而是**互补**，是体和用的关系

如果只有技术分析 → 可能买在好的价位，但买错了股票（如即将爆雷的恒大）
如果只有基本面分析 → 可能选对了股票（如苹果），但买在了最高点（小钟摆的顶端）
只有两者结合 → 在对的时间买对的股票

## 总结：你真正需要的三样东西

### 第一：两条公设 — 心法

- 价格围绕价值波动
- 钟摆式过度波动

### 第二：一个工具 — 重剑

- 趋势 + 均线

### 第三：一个原则 — 内功

- 技术分析服从基本面分析

### 需要扔掉的东西

- ❌ 头肩顶、双底、W底这些形态学 — 人脑量化，早被计算机淘汰
- ❌ MACD、KDJ、RSI这些指标 — 量化1.0，早被量化2.0打穿
- ❌ 波浪理论、江恩理论、缠论 — 玄学，是算命

### 终极感悟

> 重剑无锋，大巧不工。
>
> 当你的内功足够深厚时，你不再需要任何花哨的招式。你只需要用那把最重最钝的玄铁重剑，用最简单直接的方式就足以碾压对手。
>
> 投资也是一样。当你真正理解了基本面，当你真正看懂了一家公司的价值，技术分析自然就简化为 — 看清短期价格的钟摆运动在哪个阶段，然后在合适的时机出手。
>
> 技术分析的终点，就是回到「价格围绕价值波动」这个起点。
