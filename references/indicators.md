# 技术指标详解

## 核心工具：移动平均线（MA）— 玄铁重剑

均线是本工具的核心分析工具，是价值中枢在图表上的投影。

### 本质

均线 = 一定周期内的平均价格 = 市场对价值的平均共识 = 价值中枢

### 常用均线及含义

| 均线 | 周期 | 含义 | 用途 |
|------|------|------|------|
| MA5 | 5日 | 一周平均 | 超短期情绪，微型钟摆 |
| MA10 | 10日 | 两周平均 | 短期趋势 |
| MA20 | 20日 | 一月平均 | 短期价值中枢，小钟摆支点 |
| MA60 | 60日 | 一季平均 | 中期价值中枢，中钟摆支点 |
| MA120 | 120日 | 半年平均 | 中长期价值中枢，大钟摆支点 |
| MA250 | 250日 | 一年平均 | 长期价值中枢，超大钟摆支点 |

### 功能一：均线方向 = 价值方向

- **均线向上**（斜率为正）→ 价值在提升，「人在上山」
- **均线向下**（斜率为负）→ 价值在下降，「人在下山」
- **均线走平** → 价值稳定，「人在休息」

斜率越大，趋势越明确。

### 功能二：均线排列 = 趋势状态

**多头排列**（强势信号）：
```
价格 > MA5 > MA10 > MA20 > MA60 > MA120
```
含义：各级别价值中枢都在上升，趋势向上确认。

**空头排列**（弱势信号）：
```
价格 < MA5 < MA10 < MA20 < MA60 < MA120
```
含义：各级别价值中枢都在下降，趋势向下确认。

**震荡缠绕**：
均线交叉缠绕，无明确排列 → 趋势不明确，观望为主。

### 功能三：均线偏离度 = 钟摆位置

```
偏离度 = (当前价格 - 均线值) / 均线值 × 100%
```

| 偏离度 | 钟摆位置 | 含义 | 操作提示 |
|--------|---------|------|---------|
| > +15% | 极度偏高 | 绳子极紧，回归压力极大 | 强卖出信号 |
| +10% ~ +15% | 偏高 | 回归压力增大 | 注意风险 |
| +5% ~ +10% | 略高 | 正常偏高 | 可持有 |
| -2% ~ +5% | 中枢附近 | 适合操作 | 适合买入/做T |
| -5% ~ -2% | 略低 | 轻微偏离 | 关注买入 |
| -10% ~ -5% | 偏低 | 回归动力增强 | 买入信号 |
| < -10% | 极度偏低 | 绳子极紧，反弹动力极大 | 强买入信号 |

### 功能四：支撑与压力

- 价格在均线上方 → 均线为支撑
- 价格在均线下方 → 均线为压力
- 多条均线汇聚 → 强支撑/强压力

## 核心工具：趋势定义

### 方向判断

最朴素的定义：
- **上升趋势** = 高点递增 + 低点递增
- **下降趋势** = 高点递减 + 低点递减
- **震荡** = 高低点无规律

### 趋势强度评分（0-10分）

| 维度 | 分数 | 说明 |
|------|------|------|
| 均线排列 | 0-3 | 完美多头3分，多头2分，偏多1分 |
| 高低点 | 0-2 | 高点递增+1，低点递增+1 |
| MA20斜率 | 0-2 | 加速上行+2，上行+1 |
| 周线趋势 | 0-2 | 周线多头+2，偏多+1 |
| 价格>MA120 | 0-1 | 在半年线上方+1 |

解读：
- **8-10分**：强势上升趋势，积极做多
- **6-7分**：中等上升趋势，可适度参与
- **4-5分**：趋势不明确，谨慎操作
- **0-3分**：趋势偏弱或向下，回避

## 核心工具：量价关系

### 量价配合原则

| 价格 | 成交量 | 含义 | 操作 |
|------|--------|------|------|
| 上涨 | 放量 | 多头强势，资金流入 | 买入/持有 |
| 上涨 | 缩量 | 动能不足，上涨乏力 | 谨慎 |
| 下跌 | 放量 | 恐慌出货，空头强势 | 回避 |
| 下跌 | 缩量 | 卖压减弱，可能止跌 | 观察 |
| 横盘 | 放量 | 变盘在即 | 等待方向 |

### 量比指标

```
量比 = 当前成交量 / 过去5日平均成交量
```

- 量比 > 2：成交量显著放大
- 1 < 量比 < 2：温和放大
- 0.5 < 量比 < 1：正常
- 量比 < 0.5：明显萎缩

## 辅助参考：MACD — 均线偏离度的衍生指标

> **本质说明**：MACD的红绿柱就是价格与均线距离的另一种画法。
> 直接看均线偏离度更直观、更本质。MACD仅作为可选辅助参考。

### 计算方法

```
DIF = EMA(8) - EMA(17)    ← 短期与长期均线的差值
DEA = EMA(DIF, 9)          ← DIF的均线
MACD = (DIF - DEA) × 2     ← 红绿柱
```

参数(8,17,9)比标准(12,26,9)更灵敏。

### 简要参考

- DIF > DEA（金叉）→ 短期均线向上偏离长期均线，相当于价格在加速上升
- DIF < DEA（死叉）→ 短期均线向下偏离长期均线，相当于价格在加速下降
- 本质上和直接观察MA5与MA20的关系是同一回事

## 辅助参考：KDJ — 偏离度的另一种计算

> **本质说明**：KDJ的超买超卖就是价格偏离均线程度的另一种计算方式。
> 直接看均线偏离度更直观。KDJ仅作为可选辅助参考。

### 计算方法

```
RSV = (收盘价 - N日最低价) / (N日最高价 - N日最低价) × 100
K = RSV的移动平均
D = K的移动平均
J = 3K - 2D
```

参数(6,3,3)比标准(9,3,3)更灵敏。

### 简要参考

- J > 80：价格在近期区间的高位（相当于偏离均线过高）
- J < 20：价格在近期区间的低位（相当于偏离均线过低）
- 本质上和均线偏离度反映的是同一件事

## 综合应用：评分体系

### 评分权重（满分20分）

| 维度 | 满分 | 权重 | 核心/辅助 |
|------|------|------|----------|
| 趋势方向 | 6分 | 30% | 核心 |
| 钟摆位置（均线偏离度）| 5分 | 25% | 核心 |
| 趋势强度 | 4分 | 20% | 核心 |
| 量价关系 | 3分 | 15% | 辅助 |
| 传统指标（MACD/KDJ）| 2分 | 10% | 可选参考 |

### 强买入条件

核心条件（必须满足）：
1. 均线多头排列（至少MA5>MA10>MA20）
2. MA20方向向上
3. 价格偏离MA20在合理范围内（-5%~+5%为佳）

加分条件：
4. 高点递增 + 低点递增（标准上升趋势）
5. 量价配合（缩量回调或放量上涨）
6. 周线趋势也向上

### 强卖出条件

核心条件：
1. 均线空头排列
2. MA20方向向下
3. 价格偏离MA20过大（>+10%追高风险 或 <-10%趋势已破）

加分条件：
4. 高点递减 + 低点递减
5. 放量下跌
6. 偏离MA60超过15%
