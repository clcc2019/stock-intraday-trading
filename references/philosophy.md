# 投资哲学

## 核心理论：两条公设

### 公设一：价格围绕价值波动（人牵狗）

投资大师本杰明·格雷厄姆的比喻：

- **价值** = 一个在路上行走的人（主人）
- **价格** = 他牵着的一条狗
- **价值规律** = 脖子上的绳子

狗（价格）会忽前忽后跑来跑去：
- 时而兴奋地冲到主人前面很远 → **泡沫**（价格远高于价值）
- 时而被路边的东西吸引，远远落后于主人 → **低估**（价格远低于价值）

但是，狗永远不可能脱离主人太远。无论它跑得多偏，它最终都必须回到主人身边，并且跟随主人（价值）的方向前进。

**在图表上的体现：**
- 均线 = 价值中枢（那个牽狗的人在图表上的投影）
- 长周期均线（MA60、MA120、MA250）代表市场对价值的长期共识
- 价格偏离均线后 → 必然回归 → **均值回归原理**

### 公设二：钟摆式过度波动

价格不会温和地跟着价值走，而是呈现**钟摆式的过度波动**：

- **价值** = 钟摆的静止点
- **价格** = 钟摆本身
- **市场情绪**（贪婪/恐惧）= 驱动钟摆的外力

钟摆永远不会温柔地停在静止点：
- 涨的时候，总会涨过头（摆到最右）→ **过度乐观**
- 跌的时候，也总会跌过头（摆到最左）→ **过度悲观**

**物理定律**：钟摆摆得越远，回归的力量越大。
- 价格偏离价值越远 → 回归力（绳子的拉力）越大
- 这就是为什么暴涨之后容易暴跌，暴跌之后容易反弹

## 核心原则：顺大势，逆小势

### 嵌套钟摆模型

趋势是有级别的，钟摆是嵌套的：

| 级别 | 对应均线 | 钟摆大小 | 含义 |
|------|---------|---------|------|
| 年线 | MA250 | 超大钟摆 | 长期价值方向 |
| 半年线 | MA120 | 大钟摆 | 中长期趋势 |
| 季线 | MA60 | 中钟摆 | 中期趋势 |
| 月线 | MA20 | 小钟摆 | 短期趋势 |
| 周线 | MA5/MA10 | 微型钟摆 | 超短期情绪 |

**核心规律**：小钟摆的摆动，最终服从大钟摆的方向。

就像人在火车上，无论你往前跑还是往后跑，时间拉长一点，几乎对位置改变不起作用。把你带到另一个城市的关键因素，还是火车（大趋势）。

### 顺大势

- **大势** = 长周期趋势（周线/月线/季线方向）
- 只买均线多头排列、趋势方向向上的股票
- 确保你买的是「人在往上走」的好公司

### 逆小势

- **小势** = 短周期波动（日线/分时波动）
- 不在小钟摆冲到最高点（最贪婪）时追高
- 而是在小钟摆因短期恐慌回调到大钟摆支点（均线）附近时买入
- 这是建仓良机：长期价值在上升 + 短期价格在回调

### 买入时机的关键

你不能在日线刚开始跌的时候就买入，因为**公设二**告诉我们价格会过度波动——既然要跌，它就会跌过头。你要等那个小钟摆真正摆到底部、开始回摆的时候再买入。

## 趋势的两个维度

### 方向（趋）

最朴素的定义：
- **上升趋势** = 上涨创新高，调整不创新低（高点递增，低点递增）
- **下降趋势** = 反弹不创新高，下跌创新低（高点递减，低点递减）
- **震荡** = 既不创新高，也不创新低

### 强度（势）

势的大小决定可靠性：
- 大盘下跌时个股不跌甚至逆势上涨 → 势大
- 十几年持续上涨 vs 两三年上涨 → 前者势大
- 大盘股上涨 vs 小盘股上涨 → 大盘股需要更多资金推动，更难操纵，势更大
- 个股上涨带动板块行情 → 势更大

**势大力沉**：当你选择更大的强度参与方向交易，失败的概率更小。

## 均线 = 玄铁重剑

均线看起来又钝又慢，但蕴含着当前价值的全部信息。

### 功能一：方向 = 价值中枢的移动方向

- 均线向上 → 市场对价值的共识在抬升，基本面在改善
- 均线向下 → 市场共识在下降，基本面在恶化
- 均线走平 → 价值稳定

### 功能二：距离 = 情绪的偏离程度

- 价格远高于均线 → 过度乐观，繩子绷紧了，有回调压力
- 价格远低于均线 → 过度悲观，繩子也绷紧了，有反弹动力
- 这就是均值回归的可视化

### 为什么不需要MACD/KDJ

- MACD的红绿柱 = 价格与均线距离的另一种画法
- KDJ的超买超卖 = 价格偏离均线程度的另一种计算
- 它们是「量化投资1.0」的产物，把简单直观的道理包装得面目全非
- 直接看均线偏离度，更直观、更本质

## 技术分析的正确定位

技术分析**不是预测未来**，它只是一个仪表盘：
- 告诉你现在车速多少（价格位置）
- 发动机转速是多少（趋势强度）
- 但不能告诉你这条路通向哪里（价值方向）

技术分析能回答三个问题：
1. **判断位置（偏离程度）**：狗现在离人有多远？
2. **判断方向（趋势）**：钟摆现在朝哪个方向摆？
3. **判断情绪（极端位置）**：钟摆是否摆到了极端？

## 内功与招式

投资就像练武功：
- **基本面研究** = 内功（理解公司价值）
- **技术分析** = 招式（选择买卖时机）

致命错误：内功为零，却痴迷于学习最华麗的招式。

### 正确的投资流程

1. **技术筛选（望远镜）** — 用均线+趋势快速筛选，从几千只股票中选出大趋势向上的
2. **基本面研究（显微镜）** — 理解均线为什么在上移（业绩增长？行业景气？政策利好？）
3. **前瞻判断** — 价值提升的因素能持续多久？是初期爆发还是末期狂欢？
4. **交易决策** — 在钟摆回摆至均线附近时出手

### 记住

- 技术分析只能看到「狗」（价格），看不到「人」（价值）
- 内功为本（80%时间研究基本面），招式为辅（20%时间看图表）
- 只有两者结合，才能做到在对的时间买对的股票
